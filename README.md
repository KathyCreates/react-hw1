# React Components Demo (Vite)

Невеликий React-проєкт на Vite з двома компонентами: Button та Input.  
Компоненти приймають props і підтримують події `onClick` та `onChange`.

## Встановлення та запуск

1) Встановити залежності:
```bash
npm install
```

2) Запустити в режимі розробки:

```bash
npm run dev
```

## Компоненти:
Button: props text, type, onClick
Input: props placeholder, type, onChange
